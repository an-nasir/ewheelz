
# Product Focus

Primary goal: increase REAL EV listings and conversions.

Rules:

* Every feature must increase listings OR buyer trust
* If not → reject

Priorities:

1. Listings acquisition (scraper, WhatsApp, manual ingestion)
2. Buyer trust (deal score, battery health, verified seller)
3. Conversion (contact seller, WhatsApp click)
4. SEO pages with real traffic intent

Kill:

* unnecessary tools
* empty community features
* over-engineered dashboards

Ask before building anything:
"Does this get more listings or buyers?"

If NO → do not build
